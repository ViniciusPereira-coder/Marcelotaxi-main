# MarceloTaxi

Landing page institucional da MarceloTaxi, com foco em transporte executivo premium em São Paulo.

## Stack

- React 18
- Vite 6
- Tailwind CSS 4
- React Router 7
- Motion + Lucide

## Requisitos

- Node.js 20+
- npm 10+

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```
